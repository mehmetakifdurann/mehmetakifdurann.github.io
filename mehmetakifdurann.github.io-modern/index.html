Mobile health / Android Studio

İlaç Asistanım

İlaç Asistanım is a mobile medication management demo app designed to support elderly and chronically ill patients while reducing the tracking burden on caregivers. Built as a fully functional Android Studio project, the app includes patient and caregiver profiles, medication scheduling, stock tracking, symptom logging, missed-dose notifications, and an emergency alert flow. It was developed as a demo product with realistic in-app data, Turkish UI content, and a clean mobile-first experience to showcase the core treatment-monitoring workflow in a practical way.

Focus Medication adherence and caregiver visibility
Workflow Profiles, schedules, stock, symptoms, alerts

Scheduling / room allocation

OptiSched

OptiSched is a scheduling and room assignment solution built to simplify academic timetable planning and classroom reservations. It addresses the complexity of creating course schedules by automatically organizing class programs, assigning suitable classrooms, and helping reduce conflicts in time and space allocation. In addition, it allows instructors to reserve classrooms for quizzes, make-up sessions, or any other academic need, making the process more flexible and efficient for both administration and teaching staff.

Focus Conflict-aware timetable generation
Workflow Course scheduling, room assignment, reservations
Outcome Faster planning for academic operations
