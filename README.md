# Mehmet Akif Duran Portfolio

A modern personal portfolio website for showcasing my projects, experience, and contact details.

## Overview

This site presents my work with a strong focus on practical product thinking, clean UI design, and real-world engineering workflows. The portfolio highlights two featured case studies:

- **İlaç Asistanım** - a mobile medication management demo app built in Android Studio for elderly and chronically ill patients, with caregiver support, medication tracking, stock awareness, symptom logging, missed-dose notifications, and emergency alert handling.
- **OptiSched** - a scheduling and room assignment solution for academic timetable planning, classroom reservation, and conflict reduction.

It also includes additional project samples, experience highlights, and contact information.
