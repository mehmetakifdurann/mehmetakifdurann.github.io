This project is a modern personal portfolio website designed to present my background, skills, experience, and selected work in a clean and engaging way. The site is built to give visitors a fast understanding of my approach to software development, product thinking, and UI design, while also making it easy to review my experience and get in touch.

The main purpose of the portfolio is to create a professional online presence that reflects both technical capability and design awareness. It combines a strong visual identity with a responsive structure, smooth navigation, and a presentation style that feels modern, minimal, and easy to scan on both desktop and mobile devices.

The site is structured to highlight personal information, development experience, core technical skills, and contact details in a clear way. It is built with a focus on usability, readability, and polished visual hierarchy, so it can serve as a practical portfolio for internships, job applications, and professional networking.
